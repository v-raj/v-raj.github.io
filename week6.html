DESIGNING THE PCB

For Designing I used Eagle CAD Software

 

Download And Install Eagle

The Student Version is Free to use.

http://www.autodesk.com/products/eagle/free-download

I particularly have used the older version of eagle before its acquired by Autodesk.

I Referred Following links for Getting Started With Eagle

 

  • Eagle
  • Fab academy eagle tutorial
  • Beginner tutorial

Now  We need to add library files to the eagle

Downloaded Following Assorted libraries by our global Instructor Ohadino Meyuhas

 

Extract The folder and copy all .lbr files.

 

Now to To Library Menu Item in Eagle and open it.

Now Past all the files we downloaded earlier in the folder which opens up.

Now We are all set to start designing the PCB.

 

Open a blank eagle file.

Now go to the command panel and type : add   and press enter

This will open the component selection windows that looks like this.

Note we have 3 form factor for ATtiny44  IC select the last one above for the one we have in fablab's inventory.

The basic components of the the hello echo board are listed below:

  • ATTiny 44 as MCU x1
  • 6 Pin AVR ISP Header x1
  • FTDI Header x1
  • 20 MHz Resonator x1
  • 1 uF Capacitor x1
  • 10K Ohm Resistor x1

In addition to that I am supposed to add few more components:

  • Switch
  • LED x2
  • 499 Ohm Resistor x2

 

 

Now once all components are arranged

Go ahead and type : board  and press enter or optionally select the board icon on the left panel

 

This will popup below dialog box. press yes

 

enable the grid to enable us to draw paths in distance of 0.25 inch

 

Add the DRC rule file from the Folder we have Downloaded earlier.

Next I ran DRC : Design Rule Check and it shows above errors all are warning and can be ignored.

 

Now once its all arranged we are all set to create path.

 

Go to Tools and Select Autorouter. option and press enter

Select any one option with 100% auto routing done.

I got only one with 100% Routing complete else we might had to add it manually.

Lucky !!

 

Autorouter option is like magic to me. it calculates all possible path patterns and suggest few best of them.
